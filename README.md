# 額外頭顱
黏液科技的附加 繁體翻譯版

通過殺死相應的生物來獲得新的頭顱.<br>
你可以使用斬首之劍增加機率獲得.<br>

[官方原碼連結](https://github.com/TheBusyBiscuit/ExtraHeads)

## 下載額外頭顱
下載繁體翻譯版: [點此](https://github.com/xMikux/ExtraHeads/releases)<br>
你也可以在這裡下載官方的開發版: [開發版構建](https://thebusybiscuit.github.io/builds/TheBusyBiscuit/ExtraHeads/master/)
