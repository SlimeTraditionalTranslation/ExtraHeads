# 額外頭顱

額外頭顱是 Slimefun4 的附加，新增了新的生物頭顱到遊戲中。<br>
有一大堆的新頭顱可以透過殺死相應的來生物獲取。<br>
你可以用斬首巨劍來增加你獲得的機率。
