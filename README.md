# 額外頭顱
黏液科技的附加 繁體翻譯版

通過殺死相應的生物來獲得新的頭顱.<br>
你可以使用斬首之劍增加機率獲得.<br>

> 此為**非官方**版本, 請勿在該作者問題追蹤內回報! <br>
